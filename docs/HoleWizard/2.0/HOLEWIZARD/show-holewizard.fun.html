SHOW-HOLEWIZARD [Public Function]

Package: holewizard (Nick name(s): :hw) 

(show-holewizard :feature feature
                 :mode    action)

Show the Hole Feature Tool UICT UICT dialog.

Parameters

:feature

Hole feature to open the dialog with. Can be nil if the operation mode is :CREATE.

:mode

Operation mode. Can be :CREATE, :MODIFY, or :COPY

Returns

T for success
