3D-PRINTING.DATA-EXCHANGE [Package]

The package :3D-PRINTING.DATA-EXCHANGE (nicknamed :3DP.X) contains a LISP API and dialogs to import or export manufacturing data such as STL or 3MF with optional reverse-engineering of facet-data into analytical surfaces.

Package Contents

Dialogs

Name Summary
LOAD_MANUFACTURING_DATA Modeling load dialog extension for loading manufacturing data files (STL, 3mf).
SAVE_3MF_MODEL Save a selection of parts and assemblies as 3mf data file.

(Generic) Functions / Macros / Accessors

Name Summary
