Modeling Extension Module Auto Doc Generator

The ADGservice extension module for Modeling provides document generation and translation services (ADG - Automated Document Generator).

The installation package consists of three components which build on each other:

Component Description
ADGservice The Modeling extension module
adg Windows service to orchestrate Modeling worker processes.
adgweb ASP.net Blazor web service for interactive form based ADG job creation.
