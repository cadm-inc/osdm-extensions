RCA_SETTINGS [Public Dialog]

Package: soco-cables (Nick name(s): :rca :soco_cables)

Edit customizable module settings.

The settings dialog can be activated via the Modeling Settings user interface (File -> Settings -> Extensions -> Rigid Cable Settings ...).

See also: The Rigid Cable Advisor Customization Guide for all customization options.

Dialog

  1. Changes *rca-conn1-color* (Default color for connector1 {integer})
  2. Changes *rca-conn2-color* (Default color for connector1 {integer})
  3. Changes *rca-cable-color* (Default color for cables {integer})
  4. Changes *rca-comp-color* (Default color for components {integer})
  5. Changes *rca-lbl-rel-size* (Relative size of label with regard to total length of cable {float})
  6. Changes *rca-model-tip-dist* (Let cable protrude through the connector {boolean})
  7. Changes *rca-create-lbl-comp-feedback* (Determine if labels for components should be shown {boolean})
  8. Changes *rca-use-previous-cabledata* (Reuse previous points if creating a new cable {boolean})
  9. Changes *rca-auto-clear-offset-points* (Should point offset values be reset (i.e)
  10. Changes *rca-display-cable-path-only* (Only display cable path when adding/editing points {boolean})
  11. Changes *rca-check-illegal-cond* (Show violated rules in display window {boolean})
  12. Changes *rca-enable-cell-editing* (Allow editing of point values directly in table (no multi-selection) {boolean})
  13. Changes *rca-keep-current-part* (Keep current part when editing cable (if nil, cable becomes current part) {boolean})
  14. Reset the settings to the startup configuration.
