KEYSIGHT-MIGRATOR [Package]

Primary Lisp package of the ky_migrator Modeling extension module.

The package :KEYSIGHT-MIGRATOR (nicknamed :KYM) contains dialogs and tools to facilitate the migration of Modeling 3d and 2d models to ModelManager or Enovia.

Programming Notes

This package is not intended to be included in the :use specification of other packages due to a high probability of name conflicts. To reference a facility provided by this package, code in another package must use an explicit package prefix, e.g.

(kym:load-module-customization)
