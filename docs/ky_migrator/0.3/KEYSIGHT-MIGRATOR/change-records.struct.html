CHANGE-RECORDS [Public Structure]

  ╭─persistent-records CHANGE-RECORDS  

Package: keysight-migrator (Nick name(s): :kym)

A logger for changes to component-info objects.

The logs are cached in memory until [flush-change-records] is called. Changes can be logged by calling [log-component-change].

Slots

buffer {list}
A cached list of change-info records.
