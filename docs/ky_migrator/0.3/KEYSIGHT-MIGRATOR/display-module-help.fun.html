DISPLAY-MODULE-HELP [Public Function]

Package: keysight-migrator (Nick name(s): :kym)

Display a module help page in the browser.

(display-module-help [page])

Arguments

page {string ["Home.html"]}
Optional relative path to help page. The path is relative to the extension module's documentation/pub/html directory. If this parameter is not provided, the module documentation Modeling Extension Module Keysight Migrator is opened.

Returns

T for success; NIL otherwise
