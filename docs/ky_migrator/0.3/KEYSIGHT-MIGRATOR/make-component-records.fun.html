MAKE-COMPONENT-RECORDS [Public Function]

Package: keysight-migrator (Nick name(s): :kym)

Make and a new component-records instance and initialize it with records persisted to disk earlier (if any).

⚠️ The records file on disk may contain multiple instances of info objects for the same sysid, depending on how often the info was updated. We keep the most up-to-date info only.

(make-component-records dir)

Arguments

dir {pathname}
Absolute path to the directory containing a persisted file records.

Returns

A new instance of a component-records initialized with the content of the records file found in the given directory. If the file does not exist it is created, and the table is empty.

Related Structures

Name Summary
component-records Table of model components which are sub-types of component-info indexed by sysid.
