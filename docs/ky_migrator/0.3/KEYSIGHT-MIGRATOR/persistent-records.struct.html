PERSISTENT-RECORDS [Public Structure]

PERSISTENT-RECORDS  

Package: keysight-migrator (Nick name(s): :kym)

Base structure to persist records of some sort in a single (.lsp) file.

Files are managed in a crash-resistant way. Existing records are recovered after crash when this structure is instantiated, and new records can be added thereafter.

Slots

path {pathname}
Absolute path to the file containing the persisted records.

outstream {stream} : The stream to write new records to.
