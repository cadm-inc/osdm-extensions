COMPONENT-INFO-PERSISTED-P [Public Function]

Package: keysight-migrator (Nick name(s): :kym)

Predicate to determine if a component-info object has been persisted and contains actual sysids rather than [entity] handles.

(component-info-persisted-p arg)

Arguments

this {component-info}
The component info object to apply the predicate to.

Returns

T if the component info object is in the resolved state; NIL otherwise.
