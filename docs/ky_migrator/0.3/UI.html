The Extension Module User Interface

The functionality of this extension module is accessible from the Modeling user interface in several locations. The most used dialogs of the extension module are accessible via a command group in the Extensions tab of the Modeling ribbon bar.

  1. keysight-migrator:KEYSIGHT_DRAWING_ASSOCIATOR (Associate drawings with 3d models using a Master Extraction List)
  2. keysight-migrator:KEYSIGHT_MM_IMPORT_ASSEMBLER (Stage Modeling 2d and 3d models for ModelManager check-in)
  3. keysight-migrator:KEYSIGHT_MM_ASSISTED_CHECKIN (Assisted _ModelManager check-in of 2d/3d models in a staging directory)
  4. Help actions:
    • Display the documentation for the Keysight Migrator extension module
    • Display general module information. See also keysight-migrator:about-module

Module help and management dialogs are available in following locations:

  • File -> Settings -> Extensions -> ky_migrator Settings ...: The keysight-migrator:KYM_SETTINGS (Edit customizable settings for the ky_migrator extension module)
  • File -> Modules: Opens the Modeling module management dialog where the extension module can be (de)activated in the CADM Extensions tab by (de)selecting Keysight Migrator
  • In the ❔ command group located at the right end of the Modeling ribbon bar where the action Keysight Migrator opens the module documentation in the browser.
